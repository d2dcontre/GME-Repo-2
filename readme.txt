Groups Made Easy

Requirements:
Java JDK 7
MySQL Connector/J 5.1.22
MySQL Community Server 5.5.28
Make sure both are set in the classpath

Installation:
1) Compile all of the Java files in the "Source" folder
2) Run the .sql files 
   (create example db.sql first, then create events.sql)
3) Run the Runner class to execute the program!

NOTE to ADMIN:
In Version 1.0, Daily schedules are delivered to 
students pre-made.
